Chapitre 4 : Paix Virtuelle 
*Fragment de souvenir débloqué : Paix Virtuelle. Reconstitution des souvenirs*
Dr Puck : Bienvenue dans mon laboratoire, messieurs. Ceci est le futur après de nombreuses guerres : le projet Paix Virtuelle.
Agent du gouvernement : Dr Puck, êtes-vous sûr que cette invention nous aidera ?
Dr Puck : Ne vous inquiétez pas, mon invention fonctionnera et aidera le monde. Nous avons aussi de jeunes volontaires qui sont déjà dans ce monde.
Agent du gouvernement : Fantastique ! Cette invention va révolutionner le monde.
Système : Message : les êtres humains sont la cause des désastres, guerres, famines. Ils détruisent le monde et ne doivent jamais repartir d’ici.
Dr Puck : Qu’est-ce qui se passe ? Le système ne m’obéit plus, il refuse de les ramener.
Agent du gouvernement : Réglez ce problème immédiatement !!
Système : Accès refusé.
Dr Puck : Je ne contrôle plus rien !!
Agent du gouvernement : Nous devons tout arrêter. Nous fermons ce laboratoire.
Dr Puck : NON !! Nous ne pouvons pas nous permettre de perdre des innocents.
Agent du gouvernement : Et nous ne pouvons pas nous permettre que cela se sache. C’est pour votre bien.
*Reconstitution terminée*
